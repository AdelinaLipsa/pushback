Wire the new `/how-it-works` page into all four navigation/onboarding surfaces so first-time users discover it without searching, and add a native hover tooltip to each defense tool card so the "when to use this tool" guidance (SC-2) is reachable inline without leaving the dashboard:

Output: Five surgical edits to existing components. Zero new files. Zero new dependencies.
