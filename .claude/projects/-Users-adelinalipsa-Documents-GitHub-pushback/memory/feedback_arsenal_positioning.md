Pushback must feel like a **professional toolkit / arsenal**, not an AI chat wrapper or generic assistant.

**Why:** The user explicitly flagged this as the core brand positioning. Framing it as "AI" or "chat" undermines the product's value.

**How to apply:**
- Copy should emphasize tools, precision, control — not AI generation or automation.
- Language like "21 situation-specific tools", "the right tool for the situation", "paper trail" is on-brand.
- Avoid: "AI-powered", "battle-tested", "polite nudge", "no legal background required" — these read as generic SaaS.
- Stripe is already set up in subscription mode — keep it that way, don't change billing model.
